The BDGE7SFL data contains updates to three attributes in the BDGE7RFL file.

- DHSID: updated to include the DHS country code
- CCFIPS: updated to include the correct FIPS country code
- ADM1DHS: substituted the 9999 value for the correct Admin 1 ID numbers

Please note, the coordinates and elevation did not change between datasets BDGE7RFL and BDGE7SFL.
